# RealT Token Prototype (RTD)

This repository contains an ERC-20 token prototype developed for an academic blockchain project.

## Project Objective
The RTD token serves as a proof-of-concept demonstrating how blockchain-based tokenization can represent fractional economic ownership in real estate, inspired by the RealT platform.

## Technical Overview
- Blockchain: Ethereum Sepolia Testnet
- Consensus Mechanism: Proof of Stake
- Token Standard: ERC-20
- Smart Contract Library: OpenZeppelin
- Deployment Tool: Remix IDE

## Token Parameters
- Name: RealT Demo Token
- Symbol: RTD
